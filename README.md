# TI_1
## Лабораторная работа по криптографии. Столбцовый метод и шифр Виженера
Вариант 3. Шифрование методами перестановки и подстановки

## О проекте
Программа для шифрования и дешифрования текста на русском языке с помощью двух алгоритмов:

Столбцовый метод — перестановочный шифр  
Шифр Виженера с самогенерирующимся ключом — подстановочный шифр
- Все символы, кроме русских букв, автоматически игнорируются
- Поддержка буквы Ё
- Загрузка и сохранение .txt файлов
